\##  Overview



Version 1.0 is the foundational release of this project.



This version focuses on building a clean modular structure and implementing basic vulnerability detection logic.



---



\##  Features



\- Modular architecture

\- Security header scanning

\- Basic SQL Injection testing (payload-based)

\- Basic reflected XSS detection

\- Common port scanning

\- CLI-based execution



---



\##  Limitations



\- No crawling functionality

\- Only scans a single URL

\- Injection testing is simple payload matching

\- No form detection or parameter discovery



---

This version was built to understand:



\- Python project structuring

\- Separation of concerns

\- HTTP request handling

\- Basic vulnerability scanning logic
